# Forma Rule Library: Creative Professionals Edition

A collection of 30+ high-impact file organization rules designed for designers, developers, and content creators.

## ðŸ§¹ Cleanup & Maintenance

1.  **The "Screenshot Sweeper"**
    *   **Condition**: Name starts with "Screenshot" AND Kind is Image
    *   **Action**: Move to `~/Pictures/Screenshots/{Year}-{Month}`
    *   **Why**: Clears desktop clutter instantly while keeping history.

2.  **"DMG Destroyer"**
    *   **Condition**: Extension is "dmg" OR "pkg" AND Date Added > 7 days ago
    *   **Action**: Move to Trash
    *   **Why**: Installers are rarely needed after the first week.

3.  **"Zip Zap"**
    *   **Condition**: Extension is "zip" AND Date Added > 14 days ago
    *   **Action**: Move to `~/Downloads/Archives`
    *   **Why**: Keeps the Downloads folder fresh but recoverable.

4.  **"Temp File Triage"**
    *   **Condition**: Name starts with "temp_" OR "untitled" AND Date Modified > 2 days ago
    *   **Action**: Move to `~/Desktop/To Review`
    *   **Why**: Forces a decision on loose scratch files.

5.  **"Empty Folder Pruner"**
    *   **Condition**: Kind is Folder AND Size is 0 bytes
    *   **Action**: Move to Trash
    *   **Why**: Removes ghost folders left behind by other apps.

## ðŸŽ¨ Creative Assets (Designers/Photographers)

6.  **"Raw Photo Vault"**
    *   **Condition**: Extension is "CR2" OR "NEF" OR "ARW"
    *   **Action**: Move to `~/Pictures/Raw Imports/{Date Created}`
    *   **Why**: Automatically organizes photography shoots by date.

7.  **"PSD Parker"**
    *   **Condition**: Extension is "psd" OR "psb" AND Date Modified > 30 days ago
    *   **Action**: Move to `~/Creative Cloud Files/Archived/PSDs`
    *   **Why**: Offloads heavy working files from the active desktop.

8.  **"SVG Stash"**
    *   **Condition**: Extension is "svg" AND Name contains "icon"
    *   **Action**: Move to `~/Assets/Icons`
    *   **Why**: Centralizes vector assets for easy reuse.

9.  **"Font Finder"**
    *   **Condition**: Extension is "otf" OR "ttf" OR "woff"
    *   **Action**: Move to `~/Library/Fonts/To Install` (or a custom font folder)
    *   **Why**: Corrals downloaded fonts into one staging area.

10. **"Reference Image Collector"**
    *   **Condition**: Source URL contains "pinterest" OR "dribbble" (if metadata available)
    *   **Action**: Move to `~/Pictures/Inspiration`
    *   **Why**: Builds a mood board automatically.

11. **"Video Rush Reel"**
    *   **Condition**: Extension is "mov" OR "mp4" AND Size > 500MB
    *   **Action**: Move to `~/Movies/Rushes`
    *   **Why**: Separates heavy footage from light exports.

12. **"AE Autosave Assassin"**
    *   **Condition**: Name contains "Adobe After Effects Auto-Save"
    *   **Action**: Move to `~/Documents/Adobe/AE Autosaves`
    *   **Why**: Hides the clutter generated by Adobe apps.

## ðŸ’» Developer Tools

13. **"SQL Dump Diver"**
    *   **Condition**: Extension is "sql" OR "db"
    *   **Action**: Move to `~/Documents/Database Backups`
    *   **Why**: Secures sensitive database exports.

14. **"Log File Limbo"**
    *   **Condition**: Extension is "log" OR Name ends with ".log"
    *   **Action**: Move to Trash (after 3 days)
    *   **Why**: Logs are rarely useful after the debugging session.

15. **"JSON Junkie"**
    *   **Condition**: Extension is "json" AND Name is "package-lock" OR "yarn.lock"
    *   **Action**: Move to Trash (if found on Desktop)
    *   **Why**: Prevents accidental drag-and-drop clutter.

16. **"CSV Silo"**
    *   **Condition**: Extension is "csv" OR "xlsx"
    *   **Action**: Move to `~/Documents/Data Exports/{Current Month}`
    *   **Why**: Organizes data dumps by timeline.

17. **"Environment Variable Guard"**
    *   **Condition**: Name starts with ".env"
    *   **Action**: Move to `~/Development/Secrets` (and alert user)
    *   **Why**: Prevents accidental exposure of keys.

## ðŸ§  Smart / Contextual Rules (Future AI)

25. **"Project Grouper"**
    *   **Condition**: 5+ files with same unique project code (e.g., "P-1024")
    *   **Action**: Create Folder "P-1024" and Move files there
    *   **Why**: Auto-detects active projects.

26. **"Desktop Bankruptcy"**
    *   **Condition**: Desktop item count > 100
    *   **Action**: Move all to `~/Desktop/Old Desktop {Date}`
    *   **Why**: The nuclear option for a fresh start.

27. **"Weekend Warrior"**
    *   **Condition**: Date Added is Saturday OR Sunday
    *   **Action**: Tag with "Personal"
    *   **Why**: Separates work from life based on time.

28. **"Duplicate Detective"**
    *   **Condition**: Name contains " copy" OR " (1)"
    *   **Action**: Tag "Potential Duplicate"
    *   **Why**: Highlights accidental copies.

29. **"Large File Alert"**
    *   **Condition**: Size > 2GB
    *   **Action**: Tag "Review Storage"
    *   **Why**: Helps manage disk space.

30. **"Stale Project Archive"**
    *   **Condition**: Folder in `~/Projects` AND Date Modified > 6 months ago
    *   **Action**: Move to `~/Archive/Projects`
    *   **Why**: Keeps active workspace focused.
