# Forma Roadmap Documentation

This folder contains planning documents for future Forma releases.

## Documents

### [V2-Cloud-Storage-Integration.md](./V2-Cloud-Storage-Integration.md)
**Status:** Planning | **Target:** v2.0

Comprehensive roadmap for adding cloud storage support to Forma:
- iCloud Drive (recommended for v2.0)
- Dropbox, Google Drive, OneDrive (v2.1+)
- Architecture patterns, code examples, and difficulty assessments
- Estimated 3-4 weeks for iCloud only, 8-12 weeks for full multi-cloud
