# Forma Logo Exploration

## Brand Context

**Name Meaning**: Forma (Latin) = form, shape, structure, beauty
**Tagline**: "Give your files form"
**Attributes**: Precise, Refined, Confident
**Target**: Creative professionals who value design

---

## Logo Requirements

### Technical
- Works at 16x16px (menu bar icon)
- Works at 1024x1024px (app icon)
- Works on light and dark backgrounds
- Monochromatic core (color optional)
- Recognizable silhouette

### Conceptual
- Communicates: organization, structure, precision
- Feels: premium, minimal, sophisticated
- Avoids: playful, cluttered, generic

---

## Concept Directions

### Direction 1: Stacked Layers
**Concept**: Three horizontal bars of decreasing width, suggesting organized files/layers

```
┌────────────────────┐
│ ▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬▬  │  Long bar (top)
│   ▬▬▬▬▬▬▬▬▬▬▬▬    │  Medium bar
│     ▬▬▬▬▬▬▬▬      │  Short bar (bottom)
└────────────────────┘
```

**Pros**: Simple, scales well, clearly suggests organization
**Cons**: Similar to Linear, Stripe icons
**Variations**: Vary spacing, add subtle perspective, round ends

---

### Direction 2: Grid Cell
**Concept**: A single square with internal grid lines - structure within structure

```
┌─────────────────┐
│ ┌───┬───┬───┐   │
│ ├───┼───┼───┤   │
│ ├───┼───┼───┤   │
│ └───┴───┴───┘   │
└─────────────────┘
```

**Pros**: Universal symbol of organization, very minimal
**Cons**: Might feel too generic, similar to spreadsheet apps
**Variations**: 2x2, 3x3, asymmetric grid, one highlighted cell

---

### Direction 3: Converging Lines
**Concept**: Lines radiating inward to a focal point - chaos becoming order

```
        \  |  /
         \ | /
          \|/
    ───────●───────
          /|\
         / | \
        /  |  \
```

**Pros**: Dynamic, suggests transformation, unique
**Cons**: More complex, might not scale down well
**Variations**: One-directional (left to right), subtle animation potential

---

### Direction 4: Folder Essence
**Concept**: Abstracted folder silhouette reduced to geometric essence

```
┌─────────────────┐
│    ┌──┐         │  Tab (minimal)
│ ┌──┘  └─────┐   │
│ │           │   │  Body (clean)
│ └───────────┘   │
└─────────────────┘
```

**Pros**: Directly connects to files, familiar shape
**Cons**: Every file app uses folders, less distinctive
**Variations**: Multiple nested folders, folder with grid pattern

---

### Direction 5: The F
**Concept**: A distinctive, geometric lettermark

```
┌─────────────────┐
│   ████████      │  Top bar
│   ██            │
│   ██████        │  Middle bar
│   ██            │
│   ██            │
└─────────────────┘
```

**Pros**: Clear brand initial, very scalable
**Cons**: Less conceptual, might feel corporate
**Variations**: F made of bars, F with negative space, F as folder tab

---

### Direction 6: Nested Rectangles
**Concept**: Rectangles within rectangles - organized containers

```
┌─────────────────────┐
│ ┌─────────────────┐ │
│ │ ┌─────────────┐ │ │
│ │ │ ┌─────────┐ │ │ │
│ │ │ │         │ │ │ │
│ │ │ └─────────┘ │ │ │
│ │ └─────────────┘ │ │
│ └─────────────────┘ │
└─────────────────────┘
```

**Pros**: Elegant, suggests depth and structure
**Cons**: Could feel like a "loading" indicator
**Variations**: Offset centers, rounded corners, different sizes

---

### Direction 7: Corner Accent
**Concept**: A single corner or bracket - minimal suggestion of a frame/container

```
┌─────────────────┐
│  ┌──            │  Just the corner
│  │              │
│                 │
│                 │
└─────────────────┘

OR

│  ┌──────        │  L-bracket style
│  │              │
│  │              │
```

**Pros**: Extremely minimal, suggests framing/organization
**Cons**: Might be too abstract, hard to recognize
**Variations**: Multiple corners, corner with dot

---

### Direction 8: Alignment Guides
**Concept**: Crosshairs or alignment marks suggesting precision

```
         │
    ─────┼─────
         │
         │
```

**Pros**: Speaks to precision and design tools
**Cons**: Similar to design app icons
**Variations**: With corner marks, with grid dots

## Reference Logos

### For Inspiration

| Brand | Logo Type | Why It Works |
|-------|-----------|--------------|
| **Linear** | Stacked lines | Simple, suggests layers/lists, very scalable |
| **Notion** | N with dot | Combines letter with page metaphor |
| **Things** | Checkmark | Single meaningful shape, universal |
| **Bear** | Bear head | Character-based, warm but minimal |
| **Craft** | Abstract C | Flowing, elegant, suggests creativity |
| **Raycast** | Ray burst | Suggests speed and light |
| **Arc** | Curved line | Simple, unique, memorable |

### What to Avoid

| Type | Why |
|------|-----|
| Generic folder | Every file app uses one |
| Colorful gradient | Doesn't match brand (monochromatic) |
| Complex illustration | Won't scale to menu bar |
| Overly playful | Conflicts with "Precise, Refined, Confident" |

## Exploration Questions

1. **Which direction resonates most?** (Pick 2-3 to explore further)

2. **Should the logo be:**
   - Purely abstract (shape only)
   - Include the F lettermark
   - Be the full wordmark "Forma"

3. **What feeling matters most:**
   - Organization (structured, orderly)
   - Precision (exact, careful)
   - Transformation (chaos → order)

4. **Physical analogies that resonate:**
   - Organized desk
   - Architectural blueprint
   - Filing cabinet
   - Swiss watch mechanism
   - Geometric crystals
